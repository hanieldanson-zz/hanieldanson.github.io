Variables are containers for data and can be either numbers or words (a string). The syntax follows: 

var *nameOfVariable* = *value*
var x = 5;
var message = “Hello world!”;
var empty; // undefined variable "empty."
// Undefined is not a string or integer.
// Note: you can't create two variables with the same name.

JavaScript Objects are similar to real objects. They have properties and you can do things to the them. In JavaScript, doing something to an object can be described as using a method on an object. Create a JavaScript Object with: 

Object Literal: Most common way to create object
var myObject = {}     // An empty object
var person = {      // An object with 2 items
  name: “Bob”;
  age: 19;
}
Object Constructor: Next most common way to create object
var person = new Object ();
person.name = “Bob”;
person.age = 19;

The Constructor pattern is another way to use the object 
constructor with this, referring to the global object.
function Person(theName, theAge) {
  this.name = theName;
  this.age = theAge;
}
var bob = Person(“Bob”, 19);

The Prototype pattern can also be used as an alternative 
for creating objects. For example:
function Person() { }
Person.prototype.name = “Bob”;
Person.prototype.age = 19;
var bob = new Person(); // calls constructor to create 
bob w/ the properties name = “Bob” and age = 19

Properties are values associated with a JavaScript object. Properties can be accessed two ways: 

Dot notation: obj.propertyName 
Square bracket notation: obj[propertyName]

Accessing and Enumerating Properties on Objects using a for/in loop: 

var school = {name:”DBC”,loc:”Chicago”,cohort1:”Squirrels”}
for (var eachItem in school) {
console.log(eachItem); // Prints name, loc, cohort1
}

Dot notation is easier to read and write. Square bracket notation allows access to properties containing special characters and selection of properties using variables. 

var school = {name:”DBC”,loc:”Chicago”,cohort1:”Squirrels”}
console.log(school.name); // DBC
console.log(school[“cohort” + 1]); // Squirrels

hasOwnProperty returns a boolean value indicating if the object being called has a property with the name of the argument in parenthesis. This is useful when enumerating properties of an object in the for/in construct. 

var school - {schoolName:”DBC”};
console.log(school.hasOwnProperty(“schoolName”));

Each data property has 3 attributes:

Configurable: Specifies if it can be deleted or changed.
Enumerable: Specifies if it can be returned in for/in loop.
Writable: Specifies if it can be changed.

Every JavaScript function has a prototype property (empty by default), and you attach properties and methods on this prototype property when implementing inheritance. The prototype property is not enumerable (inaccessible in for/in loop).

An object’s prototype attribute points to the object’s “parent”—the object it inherited its properties from. The prototype attribute is normally referred to as the prototype object, and it is set automatically when you create a new object.

JavaScript Functions are defined by either a declaration or an expression. Functions are sequences of instructions that can be triggered by one line of code -- substitute repeated code with functions. 

Function Declarations:

function functionName (parameter1, parameter2) {
  ...
}

Functions can also return values. Here, a function
requires two paramaters, and returns their sum:

function doSomething(x, y) {
  return x + y;
}

Function Expression:

Function expressions are an alternative to 
function declarations:
var doSomething = function (x, y) { return x + y; }

For both methods, you can call the function
in the same way:
var sum = doSomething(5, 12);  // 5 + 12 ==> returns 17

JavaScript Methods: 
Access a method with the following syntax:
obj.methodName()

The most reliable method to see if an object is an array:
isArr = Object.prototype.toString.call(arrayiter) 
  == '[object Array]';

Number Methods:
constructor(): Returns function that created object's 
instance. By default this is the Number object.
toFixed(): Formats a number with specific # of digits 
to the right of the decimal.
toPrecision(): Defines total digits left and right
of decimal to display of a number.
toString(): Returns string representation of #'s value.

Boolean Methods:
toSource(): Returns string w/ source of Boolean object; 
you can use this string to create equivalent object.
toString(): Returns a string of "true" or "false" 
depending upon the value of the object.

String Methods:
charAt(index): Returns the character at the 
specified index.
concat(string1, string2): Combines the text of two 
strings and returns a new string.
indexOf(string): Reports the zero-based index of the 
first occurrence of the specified string in this instance.
length(): Returns the length of the string.
match(regex): Used to match a regular expression
against a string.
replace(findThisStr, replacementStr): Find a 
match between regular expression and string. Replaces 
the matched substring with a new substring.
search(str): Searches the string for a matching string or 
reg. expression -- returns the index at the match
slice(index1, index2): Extracts a section of a string and 
returns a new string.
split(delimeter): Splits a String object into an array of 
strings by separating the string into substrings.

Array Methods
concat(array): Returns a new array comprised of this array
joined with other array(s) and/or value(s).
every(): Returns true if every element in this array
satisfies the provided testing function.
filter(): Creates new array with all of the elements of 
array for which the filtering function returns true.
forEach(): Calls a function for each element in the array.
indexOf(obj): Returns the first (least) index of an
element within the array equal to the specified value.
join(): Joins all elements of an array into a string.
map(funct): Creates a new array with results of calling
a provided function on every element.
pop(): Removes last element from array and returns it.
push(object): Adds one or more elements to the end of an 
array and returns the new length of the array.
reduce(): Apply function against two values of the array 
to reduce it to a single value.
reverse(): Reverses the order of the elements of an array.
shift(): Removes 1st element from array and returns it.
slice(): Extracts section of array and returns a new array.
sort(): Sorts the elements of an array.
splice(): Adds and/or removes elements from an array.
